etoken в корпоративных сетях

Date Пн 01 Сентябрь 2014 Tags getnoo / etoken / enterprise

Для соединения с корпоративной циской можно использовать AnyConnect клиент (работает через куку по SSL). Для красноглазых имеется openconnect (в генте есть в портежах).

Наш ключик: http://www.safenet-inc.com/products/data-protection/two-factor-authentication/etoken-pro/?aldn=true

Нам понядобятся драйвера: http://www.aladdin-rd.ru/support/downloads/21493/ http://www.aladdin-rd.ru/support/downloads/29366/

Для гентов можно использовать оверлей товарища mva: http://gpo.zugaina.org/app-crypt/etoken-sac Нужен небольшой патч (25.02.2014 отписал автору, может поправит):

Вот более подробная инструкция: http://www.it-lines.ru/blogs/linux/nastrojka-etoken-v-gentoo-linux

Тут https://www.mayrhofer.eu.org/aladdin-etoken-under-linux товарищ рассказывает как можно с помощью opencs (https://www.opensc-project.org/opensc/wiki/AladdinEtokenPro#Linux) c этим делом общаться.
